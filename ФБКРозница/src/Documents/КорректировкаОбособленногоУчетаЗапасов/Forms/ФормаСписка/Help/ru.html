Корректировки обособленного учета запасов

Документ предназначен для корректировки данных при ведении обособленного учета по сделкам, менеджерам или подразделениям.

Возможность обособленного учета определяется функциональной опцией Обособленный учет себестоимости товаров по видам запасов в разделе НСИ и администрирование - Финансовый результат и контроллинг -Учет товаров и выбранным объектом для ведения обособленного учета: По подразделениям или менеджерам и/или По сделкам.

Ведение подобного учета позволяет детализировать данные о поступлениях и расходах в рамках объектов обособленного учета (сделок, менеджеров и т.д.).

Для того чтобы зарегистрировать новую корректировку обособленного учета необходимо в списке корректировок воспользоваться командой Создать.
